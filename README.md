# Simplify

Source code of an end to end user application containing user authorisation, API and database integration. 

Hosted on Azure and accessible at:  https://mos-client.azurewebsites.net 

You will be prompted to create an account. When specifying the email for the account, make sure to click 'send token'. A token will be sent to the email address, the account creation box will ask you to input this token and click 'verify'.

Uses:
Entity Framework,
Owin,
ASP.Net
